Relative distance to Cobb-Douglas efficient frontier approach - Validation Set

This page presents results for the Relative distance to Cobb-Douglas efficient frontier method with the validation set. The validation set includes only 3 EDBs (Vector, Alpine and Orion) and only a few years of data (from 2017 to 2021).

The following outputs have been considered:

  • nb_connections with a sensitivity of 1.508
  • length_circuit with a sensitivity of -1.301
